# 🎬 IMDb Movie Analysis

A simple Exploratory Data Analysis (EDA) project built using Python to explore the IMDb Movie Dataset.

The goal of this project was to clean the dataset, analyze different aspects of movies, and visualize meaningful insights using Matplotlib and Seaborn.

---

## 📊 Analysis Performed

- Distribution of IMDb ratings
- Top directors based on average IMDb rating
- Highest-rated movie genres
- Movie releases over the years
- Relationship between IMDb ratings and revenue
- Relationship between IMDb ratings and number of votes
- Correlation analysis between numerical features
- Correlation heatmap

---

## 🔍 Key Insights

- Most movies have IMDb ratings between **6 and 8**.
- IMDb Rating and Revenue show a **weak positive correlation**.
- IMDb Rating and Votes show a **moderate positive correlation**.
- Revenue and Votes have one of the strongest positive relationships in the dataset.
- Audience ratings and Metascore are moderately correlated.
